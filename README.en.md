### 🔍 AutoCompletion `[CoreELEC-IL Edition]`

</div>

---
<br>
<div align="left">

> An autocomplete add-on for content search that makes searching in Kodi smarter and more accurate.

</div>


 **Full compatibility:** Works perfectly on Kodi version 21 and above, and is fully adapted to work with the latest versions.

 **Smart Search (Clean Search):** The system automatically strips out unnecessary words from your search (like "season," "episode," "quality," "movie," etc.). For example, if you search "The Chosen One season 1," the system understands you're looking for "The Chosen One" and performs an accurate search without the noise words — in both Hebrew and English.

> 💡 **User experience:** When you start typing and use autocomplete, you might notice the text stops updating in the search bar — no need to worry! That means the system has already "caught" the term and is performing the search for you in the background, instantly.

🗂️ **Local memory:** Fast access to recent searches.

🔄 **Adapted interface:** Full RTL support (right-to-left writing direction).

⏱️ **Stability:** Timeout management to prevent freezes.

<details>

<br>
<summary><b>⚙️ Technical Specs & Improvements (CoreELEC-IL Rewrite) — Click to expand</b></summary>



The add-on is originally based on the work of Philipp Temminghoff 🌱, but underwent a comprehensive, thorough rewrite to meet modern Kodi and CoreELEC standards:

**Full compatibility with <img src="https://img.shields.io/badge/-Python%203-3776AB?logo=python&logoColor=white&label=" height="20"/>:** Fully adapted for modern work environments (starting from <img src="https://img.shields.io/badge/-Kodi%2019+-17B2E7?logo=kodi&logoColor=white&label=" height="20"/> and above), ensuring compatibility with all current versions.

**Independent search engine 🔍:** Unlike Philipp's original extension — which relied on external libraries (such as `requests` and other third-party dependencies) that suffered from a lack of updates and an outdated Python 2 codebase — the CoreELEC-IL version is built on a fully independent engine.

**"All-in-one" experience 📦:** The beauty of this extension is that everything lives under one roof. Unlike the original version, which required installing cumbersome external dependency add-ons, this version relies on the built-in libraries already installed in your Kodi.

**Cross-platform compatibility 🌐:** Thanks to its independent architecture, the add-on guarantees full support and excellent performance on any platform (📦 CoreELEC, 🤖 Android, 🪟 Windows, 🐧 Linux) or any other operating system running Kodi.

**The result ✨:** The built-in search sluggishness and the incorrect-results issue that characterized the previous version have been eliminated. The system is now lightweight, stable, and significantly faster.

**Requires a supporting skin 🎨:** The add-on requires a skin that supports custom search in order to properly display the autocomplete results (such as Estuary or skins based on it).

> ⚠️ **Important note:** In some skins, you need to make sure in the skin settings that the "AutoCompletion" option is enabled, and that the correct Provider is selected for "CoreELEC-IL."

🆕 **What's new in `v3.0.1`:** Fixed a bug in search-term cleanup and a leftover-characters issue.

### 🛡️ Safe Boot Manager `[CoreELEC-IL Exclusive]`

---

</div>
<br>

<div align="left">

> The add-on that keeps your system smooth and stable — from the moment it boots until the moment it shuts down.





Safe Boot Manager 🧠: an original, exclusive project of mine, written from scratch. The need for it arose from dealing with "lightweight" operating systems like LibreELEC and CoreELEC, known for their high sensitivity to even the smallest system changes. On these systems, multiple remote button presses or momentary load spikes during boot can lead to freezes or crashes, so I built this add-on as an active protection mechanism that ensures the system stays stable at every moment.

<br>

 ⚖️ **More importantly:** The add-on solves the familiar dilemma with "heavy" skins — no matter which skin you choose to use, Safe Boot Manager knows how to manage your streamer's resources optimally, keeping the experience smooth, fast, and stutter-free even under graphical load.

<details>

<summary><b>⚙️ Technical Specs & Improvements — Click to expand</b></summary>

<br>

**Boot protection mechanism**

🔒 Prevents unnecessary remote button presses during system boot by displaying a blocking interface until loading is complete.

🧹 Automatically cleans up temporary files on every boot, while preserving critical system files such as logs and interface files.

⏳ Automatically waits for system stability (preventing overload during library or PVR scans) before releasing the lock to the user.

**Maintaining stability 🧘**

📡 Continuous system monitoring; when a load is detected (video library scan, PVR scan, or a "Busy" window), the system automatically activates a blocking interface to prevent disruptions.

**Enhanced viewing experience 🎬**

🔙 A background-based mechanism that detects 3 presses of the "Back" button within one second. The system exits the player independently and in a controlled manner — you can see the player close smoothly. This feature saves you the hassle of hunting for the "stop" button every time you want to stop playback.

🌀 Smart loading messages that appear when a movie/show starts and when playback ends, creating a seamless visual experience.

**🔐 Security, privacy, and provider sync management**

🔗 **Smart sync:** From the moment you install the extension, it automatically scans and detects your service providers (Trakt, TorBox, AllDebrid, Real-Debrid, Premiumize, EasyNews). Instead of manually figuring out how to set up each provider, the add-on proactively offers you the provider connections in its settings, which open automatically once installation finishes. You can enable or disable each provider easily in the settings window, but you must restart Kodi for changes to take effect.

> 🔴💡 **Note:** The add-on is built to work with the POV extension and uses it to send searches to scrapers. Installing POV is recommended for the add-on to work optimally.

🥷 **Token-masking mechanism:** A unique feature that ensures no full access Token (API Token) is ever written to the log file. Only the last 4 characters are shown for identification purposes, preventing exposure of credentials when sharing logs.

✅ **Health check:** For each provider you select, the add-on automatically checks on every boot whether you're connected, and shows a clear visual indicator — connected or not connected — with a quick-connect suggestion if a Token is missing.

**🎨 User interface**

🪶 **Lightweight design:** Dynamic creation of graphical elements ensures proper operation on any skin.

🚫 **Active blocking:** A built-in mechanism that prevents remote button presses from reaching the system while the interface is active.

📊 **Progress indicator:** Displays progress percentages and process names (checking providers, cleaning temp files, waiting for system).

**🛡️ Quality assurance & development**

🔬 Over two months of intensive development and code written entirely from scratch went into this add-on. Every line went through rigorous lab testing on a Homatics 4K Box streamer to ensure perfect performance and full hardware compatibility. Every update released to the GitHub repository goes through a comprehensive, thorough performance test suite before being released to the public. This policy is designed to ensure the add-on is stable and reliable without compromise. That's why there's no urgent or aggressive update cadence — I choose an approach of absolute stability and long-term reliability over frequent changes.

🆕 **What's new in `v1.0.5`:** Translation bug fixes and settings file update.

Original development from scratch — no external dependencies, stable and fast.

</details>
